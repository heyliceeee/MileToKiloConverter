## 📝 **Conversor Miles → Kilometers (Tkinter)**

Aplicação gráfica em Python que converte milhas para quilómetros usando **Tkinter**.

## 🚀 **Descrição do Projeto**

Esta aplicação cria uma janela gráfica onde o utilizador pode:

- **Inserir um valor em milhas**
- **Clicar no botão “Calculate”**
- **Ver o resultado convertido automaticamente para quilómetros**
---

## 🧩 **Funcionalidades Principais**

- **Janela Tkinter** com título, tamanho mínimo e padding.
- **Labels informativos** para guiar o utilizador.
- **Entry** para introdução do valor em milhas.
- **Botão de cálculo** que executa a conversão.
- **Atualização dinâmica** do valor convertido no ecrã.
